# Design: Restructure `new` Command Templates

## Overview

Refactor the project template structure generated by the `ppx new` command to follow Go standard layout conventions with `cmd/` and `internal/` directories.

## Motivation

The current structure uses flat top-level directories (`module/`, `infra/`, `server/`, `shared/`). The new structure follows Go conventions with:
- `cmd/` for entry points
- `internal/` for private application code
- `config/` at root for configuration files

## File Renames

| Old Name | New Name | Location |
|----------|----------|----------|
| `init.go` | `module.go` | In each module (auth, user, post) |
| `repository.go` | `repo.go` | In modules that have it (user, post) |
| `database.go` | `mysql.go` | Moved to `internal/platform/db/` |

## Template Reorganization

### Template Directory Changes

| Old Template Path | New Template Path |
|-------------------|-------------------|
| `templates/main.go.tmpl` | `templates/cmd/api/main.go.tmpl` |
| `templates/module/*/init.go.tmpl` | `templates/module/*/module.go.tmpl` |
| `templates/module/*/repository.go.tmpl` | `templates/module/*/repo.go.tmpl` |
| `templates/infra/*.tmpl` | `templates/platform/*.tmpl` |

### Import Path Changes

All templates will have updated import statements:

| Old Import | New Import |
|------------|------------|
| `"{{.ModuleName}}/module/user"` | `"{{.ModuleName}}/internal/modules/user"` |
| `"{{.ModuleName}}/infra"` | `"{{.ModuleName}}/internal/platform"` |
| `"{{.ModuleName}}/server"` | `"{{.ModuleName}}/internal/server"` |
| `"{{.ModuleName}}/shared"` | `"{{.ModuleName}}/internal/shared"` |
| `"{{.ModuleName}}/module/contract"` | `"{{.ModuleName}}/internal/modules/contract"` |

## Decisions Made

| Decision | Choice |
|----------|--------|
| `config/` location | Keep at root level |
| `dto.go` files | Keep in each module |
| New platform dirs (`cache/`, `queue/`, `logger/`) | Skip (not created) |
| `contract/` location | Move to `internal/modules/contract/` |
| `pkg/` directory | Skip (not created) |
| Approach | Clean rewrite (breaking change) |
